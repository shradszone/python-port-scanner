# python-port-scanner
A simple multi-threaded port scanner built using Python
This tool scans a range of ports on a given IP address and displays open ports.

## Features
- Multi-threaded scanning
- Adjustable port range
- Custom timeout support
- Fast and lightweight

## How It Works
The program attempts to connect to each port in the specified range.
If a connection is successful, the port is marked as open.

## Example Output
Port 80 is open
Port 443 is open
Scanned ports in 2.02 seconds
